# kowalski7cc-profile-picture

Kowalski Dragon user profile picture

## Building

You can build the wallpapers using the provided Makefile.

```bash
sudo make install
```

Support for buildin RHEL-based package is provided by spec file.

```bash
sudo make rpm
```
